# Graphs in Computer Sciences * Graph (G) is a Vertext (V) and an Edge (E) * Vertext is an object * Edge defines the relationship * The Edge has directions, in or out

Sample Graph

  • John and Clark know each other and are friends. Relationship is two way
  • Carrie and Clark know each other, Carrie is the boss. Relationship is one way

Graph Models Relationships

  • Solves the complexity of relationships
  • Relationship can have attributes; works_with, works_for
  • Richer than SQL databases for these types of datasets
  • Imagine a friend of a friend query in SQL with join table
